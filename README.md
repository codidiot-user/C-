Welcome to my **C# Practice Repository**! 🚀

This repository contains all my hands-on practice codes, examples, and solutions while learning **C#** and **.NET**. It is organized by topics to make it easy to navigate and revise concepts.

## 📁 Repository Structure

- **BasicsInC#** → Fundamental concepts, syntax, variables, control statements, methods, etc.
- **OOPS** → Object-Oriented Programming concepts (Classes, Inheritance, Polymorphism, Encapsulation, Abstraction, etc.)
- **Collections** → Lists, Dictionaries, Arrays, HashSet, Queue, Stack and other collection types
- **LinQ** → Language Integrated Query – queries, filtering, grouping, joining, etc.
- **Problems** → Coding problems and algorithm practice
- **RealTimeProblems** → Real-world scenario-based problems and solutions

## 🎯 Purpose

- To document my C# learning progress
- To serve as a quick reference for important concepts
- To practice clean and readable C# code
- To build a strong foundation before moving to advanced topics (ASP.NET, Entity Framework, etc.)
